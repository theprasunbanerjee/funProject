# 🏏 Thala For a Reason

> *"Thala for a reason"* — an Indian cricket meme that claims MS Dhoni's legendary number 7 appears everywhere.

This fun web app takes **any name** you type and mathematically proves it sums to 7. Every. Single. Time.

---

## ✨ What It Does

- You type a name.
- The app maps each letter to a digit and sums them up.
- No matter what you type, the result is **7**.
- If the result is 7, it plays the iconic Thala music and video in the background.
- If it somehow is not 7 (it will be), it still finds a way back to 7.

---

## 🚀 How to Run

No install needed — just open `index.html` in any browser.

```
git clone https://github.com/theprasunbanerjee/funProject.git
cd funProject
# Open index.html in your browser
```
